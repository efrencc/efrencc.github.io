Navigating the Azure Interface

Home

Let’s take a few minutes to get familiarized with Azure’s online portal and organization.

When you log in, you should encounter the Home page (see Figure ##). If you ever get lost in the Azure maze, click on the triple bar on the top left and select Home to get back here. By the way, that triple bar button opens the portal menu. Remember that’s what it is called because we will mention it again.

Do not mistake Home for the Dashboard. A few words about dashboards are found at the end of this mini guide.

Home is divided into different sections; the two top ones are Azure Services and Resources. Under Azure Services you will find several options: All Resources, Subscriptions, Resource Groups, Cost Analysis, etc. Under Resources you will find a list of all the resources in that particular Azure subscription (which should be empty if you haven’t created any resource). This is a lot, let’s untangle them one by one:

Accounts and Subscriptions

By now you already have an azure account. This is likely your personal account and it is linked to your Northwestern email. Under your account, you may have different subscriptions. Each subscription has its own settings and set of services it can access. The main reason you would have different subscriptions is for management and billing purposes. Likely, you will only have one subscription.

Adding Others to your Subscription

To add others user to your subscriptions you will need to navigate to your subscription.

  1. At the Azure portal, select Subscriptions.
  2. Select the subscription you want to assign and then select Access Control.
  3. Select Add Role Assignment.
  4. Select a Role you would like the user to have. (For a list of pre-built roles, see here)
  5. In the Member tab you select which user get what roles with the subscription.

Resource Groups and Resources

The core of Azure are the resources. These are specific instances you create of Azure’s different services: AI implementations, databases, networks, virtual machines, etc. Resources can be organized into groups. These are called Resource Groups and serve as an organization device. You can, for example, set up different budgets for different resource groups. Great. Now you know basic navigation in the Azure portal and the organization of resources. To start actually using Azure’s services, we need to learn a bit more about resources and cost management. We will head there next.

Dashboards

The Dashboard is not your home, it is on a different page and is basically a place to view your Azure resources and applications from different perspectives. You can have multiple dashboards. At the beginning there is one default dashboard without much in it: a list of resources and of guides. However, you can create personalized dashboards summarizing different elements of your Azure subscription. Azure provides some handy pre-made dashboards: one showing an inventory of all resources accompanied with some statistics about them, one to track different applications, one to track SQL databases, etc.
