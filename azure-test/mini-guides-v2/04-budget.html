Budgets and Costs

It is extremely important, when using cloud services, to keep an eye on incurred charges and to set up a budget. There are some resources in Azure that will keep charging you even if you are not actively using them, which may result in unexpected costs. We recommend you erase any created resources that are easily replaceable and that you do not actively use. This will differ from resource to resource. For data storage, for example, you need to keep storing your data even if you are not actively using it. However, if you deploy a service like Azure AI Search, it will continuously charge you without you noticing and without having the need for it.

Let’s learn about Azure’s own service to help you manage costs: Cost Analysis and Budgets.

Cost Analysis

When you select a resource group, you will see a menu bar on the left (see Figure ##). Find the Cost Management subsection. There you will see different options, head to Cost Analysis. The Cost Analysis section has useful information on the charges you have incurred. The main graph shows the evolution of costs for a given period of time. You can change the period view in the options bar just above the graph. Below the graph you will also see costs split by services, location, and specific resources. This will help you track costs per resource, and identify any resource incurring costs that you are not using anymore. We highly recommend you check Cost Analysis constantly. In particular, after you have created a new resource you are not familiar with, check it every day to make sure there are no unexpected charges. To help you manage your costs, you can create different budgets. Let’s learn about them.

Budget

You can create budgets to help you track and manage costs. For that, head to the Budgets section under Cost Management. By default you won’t have any budget, so you can create one by clicking on the Add button. You can 1) specify the reset period, that is, if the budget is monthly, quarterly, or annual, 2) set the budget amount, and 3) set alert notifications for when you are close to reaching the budget (or have exceeded it). Note that a budget helps you get notifications and manage costs more effectively, however, it won’t stop the services automatically. You can still go over budget, this tool is very useful but it is more organizational than a fail-safe.

Pricing

Different Azure resources have different costs. Pricing changes by region. Note that some services are charged by the hour. For example, for Azure's AI Search, which is required if you want to use your own data using OpenAI resources, you will be charged hourly. In this link you will see a pricing table for Azure AI Search. Each resource has different pricing; hence, we recommend you search and look at the price tables before using a resource you are familiar with.
